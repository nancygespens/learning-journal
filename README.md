## Description

**Learning Journal** is a responsive personal blog-style website created to document my journey in tech, creativity, and self-growth.  
It combines thoughtful writing with a clean, modern layout, focusing on clarity, readability, and a pleasant user experience.

This project serves as both a learning archive and a creative space â€” a place to reflect on progress, challenges, and small wins along the way.


## Features

- **Responsive Layout**: Fully adaptable design for desktop, tablet, and mobile screens.
- **Dark Mode Toggle**: User-controlled light/dark theme with saved preferences.
- **Hero Article Section**: Featured post with background image and overlay.
- **Recent Posts Grid**: Card-based layout for easy content scanning.
- **View More Interaction**: Progressive content reveal for better UX.
- **About Me Page**: Dedicated section to introduce the author and background.
- **Smooth Navigation**: Simple client-side switching between pages.
- **Accessible UI**: Semantic HTML, readable typography, and focus-friendly controls.
